## Python with Node.js
The `latest` tag is currently:

- Linux Distro: alpine
- Node: 15
- Python: 3.8
- serverless: latest
- npm: latest
- yarn: stable
- pip: latest
- pipenv: latest
- poetry: latest

### Use as base image
```Dockerfile
FROM thenecromancerx/docker-python-serverless:latest
```

## Pull from Docker Hub
```bash
docker pull thenecromancerx/docker-python-serverless:latest
```
